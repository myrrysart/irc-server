This sets the Irssi's own ip to localhost:
/set dcc_own_ip 127.0.0.1

This is the command:
/dcc send <nick> <path to the file>

So on my computer when sending this file you would use:
/dcc send <nick> /home/mbonsdor/work/irc-server/dcc_test.txt

And this is how you accept:
/dcc get <nick>

Or for individual files:
/dcc get <nick> <filename>
